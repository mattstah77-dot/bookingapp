# 🧠 КОНТЕКСТ ПРОЕКТА (ОБЯЗАТЕЛЬНО ДЛЯ ВСЕХ ЗАДАЧ)

Проект — это SaaS-конструктор Telegram-ботов.

---

## 🎯 ЦЕЛЬ ПРОДУКТА

Создать систему, в которой пользователь:

1. Заходит в главный бот
2. Выбирает тип бота:

   * бот для записи (booking)
   * бот для заявок (leads)
   * в будущем: продажи (sales)
3. Отправляет токен своего бота
4. Мгновенно получает:

   * готового Telegram-бота
   * mini-app (админку)
   * полностью работающий инструмент

---

## 💰 БИЗНЕС-МОДЕЛЬ

* подписка
* партнёрская программа (основной канал роста)

---

## 🏗 ТЕКУЩАЯ АРХИТЕКТУРА

Уже реализовано:

* один backend
* Telegram webhook
* мультиарендность (разные клиенты)
* изоляция данных
* bot_token используется для идентификации
* реализован booking-бот (запись на услуги)
* есть mini-app (один, но с разными данными для клиентов)

---

## ⚠️ ТЕКУЩАЯ ПРОБЛЕМА

Система жёстко заточена под booking.
